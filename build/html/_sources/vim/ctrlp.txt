CtrlP Cheatsheet
================

Search Mode General
-------------------
<C-R>    toggle regex for searching file
<C-D>    toggle filename search or directory search
<C-N>/<C-P>  search history
<C-C>     exit
<C-Z>     mark file
<C-V>     vertical split
<C-X>     horizontal split
<C-Y>     create new file in current directory
<C-F>/<C-B>     search mode foward/backward

Editing
--------
<C-W>    clear word
<C-U>    clear line

Search Mode Specifics
---------------------

Dirs
^^^^

<CR>   Change local working dir
<C-T>  Change the global working dir
<C-V>  Change the glocal working dir for the current window
<C-X>  Change the global working dir to CtrlP's current local working dir
