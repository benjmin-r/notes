Online Diagramming¶

  • Lucidchart General diagramming, flow chart and concept maping. Polished, but without LaTeX or equation editor. PNG or PDF download format. Can be embedded into html directly.
  • Coggle Mind map. Markdown, LaTex. PDF/PNG download format
  • Draw.io General diagramming, totally free. Exported as PNG, JPG, GIF, PDF, SVG, XML, however, no links availabel in the donwloaded file. Map could be embedded in a html easily using the api it provides.
  • Creatly General diagramming. Coulb be exported as SVG with links, with some pricing.
